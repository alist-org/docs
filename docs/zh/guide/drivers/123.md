# 123网盘

https://www.123pan.com/

只需填写账户密码即可。

### 用户名

用于登录的手机号码

### 密码

用于登录的密码

### 根目录 file_id

输入要挂载的文件夹，官网URL的最后一串，如：

![123](/img/drivers/123.png)

### 使用建议

- 貌似123的API每次加载的数量有限，故如果你一个文件夹内一次性加载几百个文件，可能会报错
- 建议使用分页加载：每次一页加载50个左右问题不大
  - 使用方式：后台 ---> 站点 ---> 分页类型(自己选择) ---> 默认每页数量(自己设置一下) ---> 保存即可
